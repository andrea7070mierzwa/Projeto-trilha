Sobre o Projeto

O Conexão Carreira Tech é uma plataforma focada em fortalecer o ecossistema tecnológico de Santa Catarina. Desenvolvido como uma Single Page Application (SPA), o sistema garante uma navegação rápida e moderna.

Utilizamos LocalStorage para a persistência de dados, assegurando que as informações permaneçam salvas no dispositivo do usuário, demonstrando um modelo de arquitetura ágil para soluções de RH Tech.
